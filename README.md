Project Outline (Based on Proposal Requirements)

- User Persona: Anyone interested in history of space travel; general audience.

- Basic premise: Use an interactive map to trace the journeys of astronauts from birthplace to space (and back!). From the first manned space flights in the 1960s to the lunar ventures of future Artemis missions, watch a series of small steps turn into giant leaps for all mankind.
